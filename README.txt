ICS 32 Distributed Social Messenger

Overview
This project is a distributed social messenger application built using Python and Tkinter. It allows users to send and receive messages, manage contacts, and automatically retrieve new messages.

Features
- Send and receive messages with mutlple contacts
- Manage contacts
- Automatically retrieve new messages
- Display message history
